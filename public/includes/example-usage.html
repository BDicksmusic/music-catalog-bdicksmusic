How to Use Modular Templates

Method 1: JavaScript API


// Load both templates
await TemplateLoader.loadAllTemplates();

// Or load individually
await TemplateLoader.loadNavigation(); // Injects at top of body
await TemplateLoader.loadFooter();     // Injects at bottom of body

// Custom injection targets
await TemplateLoader.loadNavigation('#header-container', 'beforeend');
await TemplateLoader.loadFooter('#footer-container', 'afterbegin');

Method 2: Auto-load with Data Attributes


<!-- Add these elements to auto-load templates -->
<div data-auto-load-nav="body" data-nav-position="afterbegin"></div>
<div data-auto-load-footer="body" data-footer-position="beforeend"></div>

Method 3: Manual Injection


// Get template HTML and inject manually
const navHtml = await TemplateLoader.fetchTemplate('navigation');
document.getElementById('my-nav-container').innerHTML = navHtml;

Template Files

  • /includes/navigation.html - Complete navigation from composition.html
  • /includes/footer.html - Complete footer from index.html
  • /includes/template-loader.js - JavaScript loader module
